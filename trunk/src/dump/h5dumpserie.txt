"Dumps two dimensional datasets of simple datatype and one dimensional\n"
"datasets of compound datatype as a space separated table.\n"
"If more file/datasets (DATAs) are given the output tabel will be merged\n"
"row-wise. If the number of rows in multi DATA output differs, <nan> will\n"
"be appended to too short DATAs.\n"
"\n"
"Usage:\n"
"  h5dumpserie [options] DATA [DATA] ...\n"
"    DATA: FILENAME/DATASET[:COLUMNS]\n"
"      FILENAME: <dir/to/hdf5/file/filetodump.h5>\n"
"      DATASET: <dir/to/dataset/in/hdf5file/datasettodump>\n"
"      COLUMNS: COL|RANGE[,COL|RANGE]... (dump all columns if not given)\n"
"        COL: column/member number in dataset to dump (starting with 1)\n"
"        RANGE: [BEGINCOL]-[ENDCOL] dump columns from BEGINCOL to ENDCOL\n"
"          (use 1/last if not given)\n"
"    options:\n"
"      -h, --help: show this help\n"
"      -c <string>: use <string> as comment in header (Default '#')\n"
"      -s: suppress header\n"
"      -q <quote>: use <quote> to quote strings in output (Default '\"')\n"
"      -n <nan>: use <nan> for 'not a number' in output (Default 'nan')\n"
"\n"
"Example:\n"
"  h5dumpserie dir/test1.h5/grp1/grp2/mydata:1,3,5-,2 dir/test1.h5/data:-4\n"
